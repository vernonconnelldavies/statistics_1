In this repository there are 2 files: the CSV file with the real estate data and the R script that I submitted to my university on linear regression with regard to real estate prices.

House Price Regression Analysis (R)

Linear regression analysis on a house price dataset. Uses backward elimination to identify the most parsimonious model, finding that size, location and condition are the key predictors of price (adjusted R-squared of 0.996). Validates the final model with predictions on both new and existing data points.
Libraries used: tidyverse, dplyr, broom, ggfortify.
